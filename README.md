# Component Selection for FANOVA Models

Code for the numerical studies and real-data analyses in the paper:

**Component Selection for Functional ANOVA Models**

(Maintainer: Youjie Zeng, yjzeng@fzu.edu.cn)


## Repository structure

- `FANOVAModels/`: implementation of GPANOVA and SSANOVA
- `benchmarks/`: competing methods, including MARS, COSSO, ACOSSO, BSSANOVA
- `simulation/`: simulation studies
- `simulation/additive/`: additive model
- `simulation/interaction/`: interaction model
- `real/`: real-data experiments
- `real/datasets/`: datasets (Ozone, Boston, Tecator, JFK, Parkinson, and Pumadyn)
- `real/splits/`: 10-fold cross-validation splits (Ozone, Boston, Tecator) and 50 random splits (JFK, Parkinson, Pumadyn)

## Path configuration

Before running the scripts, set `root_path` in each script to the local path of this repository. For example:

```python
root_path = '/Users/yjzeng/Desktop/FANOVA/github_code/' # This is our project path
```

# Data generation

The .pkl files are not included in the repository. They can be generated by running:

```bash
python simulation/additive/make_data.py
python simulation/interaction/make_data.py
```

The simulation data will be stored in the path:

- `simulation/additive/data/`
- `simulation/interaction/data/`


For real data, the 10-fold cross-validation splits and random splits can be generated by running:

```bash
python real/boston_splits.py
python real/ozone_splits.py
python real/tecator_splits.py
python real/parkinson_splits.py
python real/pumadyn_split.py
python real/jfk_splits.py
```

The real data splits will be stored in the path:
- `/real/splits/`



## Running simulation

Proposed methods:

```bash
python simulation/additive/proposed_fit.py
python simulation/interaction/proposed_fit.py
```

Benchmark methods:

```bash
Rscript simulation/additive/benchmarks_fit.R
Rscript simulation/interaction/benchmarks_fit.R
```

## Run real data analysis

Proposed methods:

```bash
python real/ozone.py
python real/boston.py
python real/tecator.py
python real/jfk.py
python real/parkinson.py
python real/pumadyn.py
```

Benchmark methods:

```bash
Rscript real/ozone_baselines.R
Rscript real/boston_baselines.R
Rscript real/tecator_baselines.R
Rscript real/jfk_baselines.R
Rscript real/parkinson_baselines.R
Rscript real/pumadyn_baselines.R
```


## Output

Results are saved under:

- `simulation/additive/`
- `simulation/interaction/`
- `real/` # average results
- `real/results` # all results.

## Notice

> - Intermediate `.pkl` files are not included because they can be generated from the provided scripts. 
> - Users must specify the correct root_path in their own PC.
> - The Baysian method, BSSANOVA, is much time-consuming.
